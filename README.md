# Learning C# 🚀

This repository contains my personal journey through **C# 14** and **.NET 10**. Here, I document my progress in software engineering fundamentals, focusing on clean code, modern syntax, and logic exercises.

## 📂 Repository Structure

The project is divided into two main sections: theory-focused fundamentals and practical activities.

### 🏛️ 01 - Fundamentals of C#

This folder contains isolated exercises to master the core language syntax.

* `01-hello-world.cs`: Introduction to Top-Level Statements.
* `02-variables-types.cs`: Exploration of primitive types and strong typing.
* `03-statements-expressions.cs`: Logic structures and C# expressions.
* `04-input-user.cs`: Capturing and handling user input from the console.
* `05-operators.cs`: Arithmetic operations and operator precedence.
* `06-if-else-statements.cs`: Decision-making flow and conditional logic.
* `07-loops-basics.cs`: Iteration structures (for, while, foreach).
* `08-comparison-logic.cs`: Boolean logic and data comparison.

### 🛠️ 02 - Activities

Practical challenges where I combine multiple fundamentals to solve real-world problems.

* **01-budget-validator.cs**: A project budget calculator that uses `double.TryParse` for safety, arithmetic operators for cost calculation, and complex `if-else` chains to determine financial feasibility.

---

## 🛠️ Tech Stack & Concepts

* **Language:** C# 14
* **Framework:** .NET 10
* **Key Concepts Applied:**
* **Null Safety:** Using `??` and null-coalescing operators to prevent `NullReferenceException`.
* **Data Parsing:** Safe type conversion with `TryParse`.
* **Clean Code:** Meaningful variable naming and structured documentation (XML Comments).

## 🚀 How to Run

1. Ensure you have the **.NET 10 SDK** installed.
2. Clone the repository:
```bash
git clone https://github.com/jamerson-mt/learning-csharp.git

```


3. Navigate to the file folder and run:
```bash
dotnet run filename.cs

```
